# ChatGPT File Downloader Extension

Расширение для браузеров Chrome / Chromium / Яндекс.Браузер.  
Позволяет автоматически получать прямую ссылку для скачивания файлов из ChatGPT.

---

## Содержание репозитория

- `manifest.json`
- `background.js`
- `content.js`
- `popup.html`
- `popup.js`

---

## Установка (бесплатно)

1. Скачайте ZIP архива репозитория:

[Download Extension](https://github.com/Rejep06/chatgpt-file-downloader/archive/refs/heads/main.zip)

2. Распакуйте ZIP в удобную папку.

3. Откройте страницу расширений в браузере: chrome://extensions или edge://extensions

4. Включите **Developer mode (Режим разработчика)** в правом верхнем углу.

5. Нажмите **Load unpacked (Загрузить распакованное расширение)**.

6. Выберите папку с распакованными файлами.

7. Готово! Расширение установлено и готово к работе.

---

## Использование

1. Откройте ChatGPT и создайте файл для скачивания.
2. Нажмите на ссылку.
3. Расширение перехватит запрос, извлечет `download_url` и покажет кнопку **Скачать напрямую**.
4. Нажмите кнопку для прямого скачивания файла.
